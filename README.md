# primerReact
Primer proyecto realizado en React

## Comandos
Ejecutamos los siguiente comandos en la terminal:

1. Inicializa un proyecto de npm.
``` powershell
npm init -y
```

2. Instala en nuestro proyecto la libreria React y React-DOM.
``` powershell
npm install react react-dom
```

3. Instala Babel el cual nos permite hacer nuestro c√≥digo en JavaScript de versiones superiores en navegadores viejos.
``` powershell
npm install @babel/core babel-loader @babel/preset-env @babel/preset-react --save -dev
```
